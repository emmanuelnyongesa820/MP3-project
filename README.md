Python MP3 Player

Overview

This project is a simple MP3 Player developed in Python using Tkinter for the graphical user interface (GUI) and Pygame for audio playback.

The application allows users to load MP3 files and control playback through a user-friendly interface.

Features

- Load MP3 audio files
- Play selected tracks
- Pause playback
- Resume playback
- Stop playback
- Display currently loaded track

Technologies Used

- Python 3
- Tkinter (GUI)
- Pygame (Audio Playback)
- OS Module

Installation

1. Clone the repository:

git clone <repository-url>

2. Navigate to the project directory:

cd mp3-player

3. Install Pygame:

pip install pygame

Running the Application

Run the following command:

python main.py

Usage

1. Click Load Track to select an MP3 file.
2. Click Play to start playback.
3. Click Pause to temporarily stop playback.
4. Click Resume to continue playback.
5. Click Stop to end playback.
